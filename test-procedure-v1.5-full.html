ArcadiaGuard v1.5 — Procédure de test complète

Mode confrontation 2 groupes — 2-3 personnes par groupe. Chaque groupe exécute tous les tests indépendamment. À la fin, on compare les deux JSON exportés pour identifier les divergences (= bugs reproductibles vs cas bords).
  • OK = fonctionne comme attendu. KO = ne fonctionne pas / crash / comportement inattendu (écris le détail dans le commentaire). SKIP = pas testable (mod absent, hors scope, déjà KO en amont).
  • Chaque groupe renseigne son nom + membres en haut avant de commencer.
  • Cliquer "Export JSON" à la fin → fichier téléchargé. Envoyer les 2 JSON pour confrontation.
