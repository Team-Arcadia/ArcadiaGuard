🎮 Tout passe par l'interface graphique (GUI).
Ouvrez le GUI avec /ag gui — toute la configuration se fait là : création de zone, flags, membres, debug mode. Aucune commande /zone flag set n'est utilisée dans ces tests.

🐛 Mode Debug (essentiel) : dans ZoneListScreen → footer → bouton "Debug". Quand activé, un indicateur vert "Debug mode ACTIF" s'affiche dans le header. Effet : votre compte OP est soumis aux flags comme un joueur normal → vous pouvez tester seul, sans deuxième compte.

📋 Workflow standard pour chaque flag :
1. /ag gui → sélectionner testzone → Détails
2. Colonne Flags → bouton + → rechercher le flag → confirmer
3. Clic sur le flag pour le basculer OFF (rouge)
4. Échap pour fermer, puis activer le Mode Debug
5. Effectuer l'action en jeu et vérifier le résultat
6. Revenir dans Détails → bouton ✕ à côté du flag pour le réinitialiser
